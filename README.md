# Attack-on-Titan-Review-Sentiment-Analysis
Sentiment classification of anime reviews using TF-IDF feature extraction with Logistic Regression and Naive Bayes.  This project investigates how well common sentiment classification models can identify positive and negative sentiment in anime reviews from MyAnimeList (MAL).
